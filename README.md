**Minicurso de Desenvolvimento Web – SEPEX 2025**

## **Projeto Base para Cadastro de Clientes (HTML + CSS + JS + Firebase)**

Este repositório contém o **projeto base** utilizado no minicurso de **Desenvolvimento Web**, ministrado no evento **SEPEX 2025**.
O objetivo é fornecer aos alunos um ponto de partida simples, organizado e totalmente funcional para desenvolver um sistema web com cadastro, listagem, edição e exclusão de clientes — tudo em **uma única página**, com integração ao **Firebase Firestore**.

**Sobre o Minicurso**

O minicurso “Fundamentos do Desenvolvimento Web” foi realizado em 03/12/2025, das 19h às 21h30.

O objetivo não foi apenas ensinar teoria, mas sim **construir um projeto completo, simples e real**, para que cada aluno pudesse sair da aula com domínio das principais etapas do desenvolvimento de uma aplicação web moderna.
